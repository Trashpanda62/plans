Harvest Right Large · one-time setup, then the per-batch routine

One-time setup

🔌

Power

Its own 20-amp outlet (NEMA 5-20). A 15-amp adapter ships in the box, but don’t share the circuit with the food-truck load.

📍

Placement

Level, sturdy surface with air gaps around it. Room 45–75°F. Pump beside it; run the drain line into a jar.

🛢️

Vacuum pump

Oil pump: fill to the line before first run, change oil every ~20–30 batches. Oil-free: nothing to fill.

▶️

Break-in run

Run one empty cycle first so the software calibrates, then start real food.

Every raspberry batch

1

Pre-freeze 24 h

Rinse & inspect (drop mushy/moldy ones), spread whole berries one layer on the trays, and freeze solid in a regular freezer for 24 h first. High-sugar berries freeze-dry cleaner and faster this way.

2

Load a single layer

One even layer per tray, no piling. A full Large load is roughly 10–18 lb fresh across the trays. Slide them in fast so they don’t thaw.

3

Start: “Frozen” + extra dry

Tell the machine the food is already frozen. Raspberries are high-sugar, so add extra dry time — start at +24 h. Better too dry than tacky. Fruit dry temp stays ~120–125°F.

4

Run ~20–30 h

Four-plus trays of berries run roughly 20–30 h total. Let it finish the whole cycle — don’t cut it short.

5

Test before unloading

Pull one from the center of the thickest tray and snap it — it should shatter, be room-temp, with no cold/bendy core and no gumminess. Any stickiness → add 2–4 h extra dry and re-test.

6

Package immediately

Freeze-dried fruit pulls moisture from the air within minutes. Go straight into mylar + oxygen absorber, heat-seal, lot = date — then log the batch (step 6 of the safety plan).

Yield reality: ~10 lb of fresh raspberries dries down to only ~1–1.5 lb — that’s why a 1 oz pouch can carry an $8–9 price. Weigh in and out every batch so we know real cost per pouch.

1-page food-safety plan

1. SOURCE — our own farm raspberries; harvest clean, chill promptly.
2. WASH & INSPECT — gentle potable-water rinse; discard any mushy/moldy berries. (control step)
3. FREEZE-DRY — run full cycle to completion.
4. VERIFY DRY — must be crisp/brittle all the way through (log water activity < 0.6 if we get a meter).
5. PACKAGE — food-grade mylar pouch + oxygen absorber, heat-sealed. Lot code = date.
6. RECORD — one line per batch: date, lot, weight in/out, pass-fail on crispness.
